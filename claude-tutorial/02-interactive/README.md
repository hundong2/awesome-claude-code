# 02단계: 인터랙티브 모드 - 슬래시 명령어 & 단축키

## 이 단계에서 배울 것

Claude Code의 인터랙티브 REPL에서 사용하는 모든 슬래시 명령어와 단축키를 익힙니다.

---

## 2-1. 정보 확인 명령어

```
/help          → 사용 가능한 모든 명령어 목록
/status        → 버전, 모델, 계정, 연결 상태
/cost          → 이번 세션 토큰 사용량 & 비용
/usage         → 플랜 한도 및 레이트 리밋 상태
/doctor        → 설치 상태 점검
```

**실습**: `claude` 실행 후 위 명령어를 하나씩 입력해보기

---

## 2-2. 컨텍스트 관리 (핵심!)

```
/clear         → 대화 기록 초기화 (새로 시작)
/compact       → 대화를 압축해서 컨텍스트 절약
/compact "핵심 내용만 유지해줘"  → 특정 지시와 함께 압축
/context       → 컨텍스트 사용량 시각화 (컬러 그리드)
```

**언제 쓰나?**
- `/clear`: 완전히 새 주제로 바꿀 때
- `/compact`: 같은 주제를 계속하되 컨텍스트가 꽉 찰 때
- `/context`: 컨텍스트가 얼마나 남았는지 확인할 때

---

## 2-3. 세션 관리

```
/rename 이름    → 현재 세션에 이름 붙이기
/resume        → 이전 세션 목록에서 선택
/export        → 대화 내용을 파일로 내보내기
/copy          → 마지막 응답을 클립보드에 복사
```

---

## 2-4. 설정 & 도구

```
/model         → 모델 변경 (대화 중에도 가능!)
/config        → 설정 UI 열기
/permissions   → 권한 규칙 확인/수정
/memory        → CLAUDE.md 메모리 파일 편집
/vim           → vim 스타일 입력 모드 활성화
/theme         → 색상 테마 변경
```

---

## 2-5. 핵심 단축키

### 필수 단축키

| 단축키 | 동작 |
|--------|------|
| `Ctrl+C` | 현재 생성/입력 취소 |
| `Ctrl+D` | 세션 종료 |
| `Ctrl+L` | 화면 지우기 (기록은 유지) |
| `Shift+Tab` | 권한 모드 순환 (default → acceptEdits → plan) |
| `Ctrl+R` | 이전 명령어 검색 |

### 편집 단축키

| 단축키 | 동작 |
|--------|------|
| `Ctrl+K` | 커서 위치부터 줄 끝까지 삭제 |
| `Ctrl+U` | 줄 전체 삭제 |
| `Alt+B` | 단어 단위 뒤로 이동 |
| `Alt+F` | 단어 단위 앞으로 이동 |

### 고급 단축키

| 단축키 | 동작 |
|--------|------|
| `Option+T` (Mac) | Extended Thinking 토글 |
| `Option+P` (Mac) | 모델 전환 |
| `Esc+Esc` | 마지막 응답 되감기 (rewind) |
| `Ctrl+G` | 텍스트 에디터에서 입력 |

## 2-6. 멀티라인 입력 & 파일 참조

### 멀티라인 입력

```
# 방법 1: 백슬래시 + Enter
안녕하세요\
두 번째 줄입니다

# 방법 2: Option+Enter (macOS)
# 방법 3: Shift+Enter (WezTerm, Ghostty, Kitty에서)
```

### 파일 참조 (@)

```
# 파일 내용을 컨텍스트에 추가
@README.md 이 파일에 대해 설명해줘

# 여러 파일 참조
@src/main.py @tests/test_main.py 이 두 파일의 차이점을 설명해줘
```

### 빠른 입력 모드

```
# /로 시작 → 명령어/스킬
/help

# !로 시작 → bash 직접 실행
!ls -la

# @로 시작 → 파일 경로 자동완성
@./claude-tutorial/
```
